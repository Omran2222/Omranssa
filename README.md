## Features

- **Bilingual (AR/EN)** — Full Arabic/English language toggle with RTL support
- **Dark/Light Theme** — Persistent theme toggle saved to localStorage
- **Responsive Design** — Optimized for all screen sizes
- **Animated Particles** — Interactive canvas background
- **Typing Effect** — Animated role titles in both languages

## Tech Stack

- Firebase Hosting
- Google Fonts: Inter, Fira Code, Tajawal (Arabic)
- Font Awesome 6.5.1
